## How to Contribute

Interested in contributing to SynthShape? Great! Here's how:

1. Fork the repository
2. Create a feature branch (`git checkout -b amazing-feature`)
3. Make your changes
4. Commit your changes (`git commit -m 'Add some amazing feature'`)
5. Push to the branch (`git push origin amazing-feature`)
6. Open a Pull Request

## Code of Conduct

We're committed to providing a welcoming and inclusive experience for everyone. Please respect other contributors and maintain a positive environment.
